
# El Secreto de las Orquídeas - Aventura Interactiva

Historia interactiva basada en un cuento original desarrollado para el curso de Web. El objetivo es explorar la narrativa no lineal a través de leyendas místicas sobre orquídeas.

## Estructura del Proyecto
- `index.html`: Punto de entrada e introducción.
- `/leyendas`: Contiene las tres historias principales (Chise, Ari y Yoko).
- `/final`: Contiene el desenlace de la historia y las referencias.
- `/images`: Carpeta con los recursos visuales (Oncidium, Phalaenopsis, Maxillaria).

## Tecnologías Utilizadas
- HTML5 Semántico.
- Uso de rutas relativas para navegación y absolutas para recursos.

## Etiquetas Semánticas Destacadas
Para este proyecto se utilizaron etiquetas de HTML5 para mejorar la accesibilidad y estructura:
- `<main>`, `<article>`, `<section>`, `<aside>`, `<nav>`, `<header>`, `<footer>`.
- `<figure>` y `<figcaption>` para las imágenes.
- `<progress>` para mostrar el avance de la lectura.
- `<details>` y `<summary>` para secciones interactivas.
- `<time>` para fechas históricas.
